## :key: How to download the project

```bash
# clone the repo:
$ git clone https://github.com/Regis-Oliveira/igniteGoFinancesRN.git

# open the project:
$ cd igniteGoFinancesRN

# install the dependencies:
$ yarn install

# execute with:
$ yarn android / yarn ios
```
